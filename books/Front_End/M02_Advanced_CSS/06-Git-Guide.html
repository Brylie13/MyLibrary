Wireframing

Wireframing is like sketching the basic layout of a web page before adding design elements. It helps you decide where things go on the page.

Example:




    
Header

    
Main Content

    
Footer

CSS:


.wireframe {
    display: grid;
    grid-template-rows: auto 1fr auto;
}

header, section, footer {
    padding: 20px;
    border: 1px solid black;
}

Responsive Design

Responsive Design ensures that your web page works on all devices like phones, tablets, and desktops. You can use media queries in CSS to change the layout based on screen size.

Example:



This text will change color based on the screen size.

CSS:


p {
    color: black;
}

@media (max-width: 600px) {
    p {
        color: blue;
    }
}

Pseudo-classes

Pseudo-classes are special states of elements, like when a user hovers over a link. You can use pseudo-classes to apply different styles in these states.

Example:


Hover over me

CSS:


.link {
    color: red;
    text-decoration: none;
}

.link:hover {
    color: green;
    text-decoration: underline;
}

CSS Variables

CSS Variables are reusable values you can define in one place and use throughout your CSS. They help you keep your design consistent and make updates easier.

Example:



Welcome to My Website

CSS:


:root {
    --main-color: #3498db;
    --padding-size: 20px;
}

h1 {
    color: var(--main-color);
    padding: var(--padding-size);
}
