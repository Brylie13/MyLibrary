Server-Side APIs and Basic Algorithms

This project explains basic tasks using Server-Side APIs such as fetching data from an API, parsing JSON, and deconstructing parameters. It also covers some basic algorithms like isPalindrome, factorial, and titleCase. The explanations are kept simple, and the examples are written in HTML for easier understanding.

Create Fetch Request

The fetch API is used to get data from a server. It sends a request to a URL and waits for a response. Here’s a simple fetch request example to retrieve data: 


fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Parse JSON

JSON (JavaScript Object Notation) is a format used to send and receive data. When we fetch data from an API, it often comes in JSON format. We need to parse it to use it in our code: 


const jsonString = '{"name": "John", "age": 30}';
const data = JSON.parse(jsonString);
console.log(data.name); // Output: John

Demo Dynamic Fetch

You can use the fetch API dynamically, allowing you to make requests based on user input or other dynamic factors. For example, fetching user information dynamically: 


const userId = '123';
fetch(`https://api.example.com/users/${userId}`)
    .then(response => response.json())
    .then(user => console.log(user.name));

Deconstruct Parameters

Deconstructing parameters means breaking down an object into its individual parts. It is useful when working with API data. Here’s an example: 


const user = {name: 'Alice', age: 25};
const { name, age } = user;
console.log(name); // Output: Alice
console.log(age);  // Output: 25

Fetch GitHub Issues

GitHub has its own API to get issues from a repository. You can fetch GitHub issues like this: 


fetch('https://api.github.com/repos/username/repo-name/issues')
    .then(response => response.json())
    .then(issues => console.log(issues));

isPalindrome Algorithm

A palindrome is a word or phrase that reads the same forward and backward. This function checks if a given string is a palindrome: 


function isPalindrome(str) {
    const cleaned = str.toLowerCase().replace(/[^a-z0-9]/g, '');
    return cleaned === cleaned.split('').reverse().join('');
}

console.log(isPalindrome('madam')); // Output: true

Factorial Algorithm

The factorial of a number is the product of all positive integers up to that number. Here’s how you calculate the factorial: 


function factorial(n) {
    if (n === 0) return 1;
    return n * factorial(n - 1);
}

console.log(factorial(5)); // Output: 120

titleCase Algorithm

The titleCase function capitalizes the first letter of each word in a string: 


function titleCase(str) {
    return str.toLowerCase().split(' ').map(word => 
        word.charAt(0).toUpperCase() + word.slice(1)).join(' ');
}

console.log(titleCase('hello world')); // Output: Hello World
