Web APIs Overview

This project explains the basics of Web APIs using simple HTML examples. You will learn how to use event listeners, timers, data attributes, local storage, and some basic functions.

1. Creating and Appending Elements

Use Web APIs to create new HTML elements and add them to the page:

        
<div id="container"></div>

<script>
  const newElement = document.createElement("p");
  newElement.textContent = "This is a new paragraph!";
  document.getElementById("container").appendChild(newElement);
</script>

2. Event Listeners

Add event listeners to HTML elements to respond to user actions:

        
<button id="myButton">Click Me!</button>

<script>
  const button = document.getElementById("myButton");
  button.addEventListener("click", function() {
    alert("Button was clicked!");
  });
</script>

3. Timers and Intervals

Create a countdown timer using setInterval:

        
<button id="startTimer">Start Countdown</button>
<p id="countdown"></p>

<script>
  document.getElementById("startTimer").addEventListener("click", function() {
    let timeLeft = 10;
    const timer = setInterval(function() {
      document.getElementById("countdown").textContent = timeLeft + " seconds remaining";
      timeLeft--;

      if (timeLeft < 0) {
        clearInterval(timer);
        document.getElementById("countdown").textContent = "Time's up!";
      }
    }, 1000);
  });
</script>

4. Data Attributes

Use data attributes to store extra information inside HTML elements:

        
<button data-color="blue" id="colorButton">Get Button Color</button>

<script>
  const button = document.getElementById("colorButton");
  button.addEventListener("click", function() {
    const color = button.getAttribute("data-color");
    alert("The button's color is: " + color);
  });
</script>

5. Local Storage

Store data in the browser's local storage to keep it even after the page refreshes:

        
<input type="text" id="nameInput" placeholder="Enter your name">
<button id="saveName">Save Name</button>
<p id="displayName"></p>

<script>
  document.getElementById("saveName").addEventListener("click", function() {
    const name = document.getElementById("nameInput").value;
    localStorage.setItem("savedName", name);
    document.getElementById("displayName").textContent = "Saved Name: " + name;
  });

  window.onload = function() {
    const savedName = localStorage.getItem("savedName");
    if (savedName) {
      document.getElementById("displayName").textContent = "Saved Name: " + savedName;
    }
  };
</script>

6. Example Functions

Sum Array

A simple function to add all numbers in an array:

        
<script>
  function sumArray(arr) {
    return arr.reduce((total, num) => total + num, 0);
  }

  const result = sumArray([1, 2, 3, 4]);
  console.log(result); // Outputs 10
</script>

Log Even Numbers

A function that logs only even numbers from an array:

        
<script>
  function logEvenNumbers(arr) {
    arr.forEach(num => {
      if (num % 2 === 0) {
        console.log(num);
      }
    });
  }

  logEvenNumbers([1, 2, 3, 4, 5, 6]); // Outputs 2, 4, 6
</script>

Countdown

A simple countdown function:

        
<script>
  function countdown(num) {
    let timer = setInterval(function() {
      console.log(num);
      num--;
      if (num < 0) {
        clearInterval(timer);
        console.log("Time's up!");
      }
    }, 1000);
  }

  countdown(5); // Starts countdown from 5
</script>
