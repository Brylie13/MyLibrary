Conditional Statements

Conditional statements let us run different code depending on conditions. For example, we use if and else to check if something is true or false.

Iteration (Loops)

Iteration means repeating something over and over. In JavaScript, we use for or while loops to run the same code many times.

Functions

A function is a block of code that we can use again and again. We define it with function and call it by its name.

Scope

Scope refers to where a variable can be used in your code. Variables can have global or local scope.

Object and This

An object is a collection of data and actions (called methods). The this keyword refers to the object it belongs to.
